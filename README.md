# PLN - Análise Sintática com spaCy

Projeto simples de Processamento de Linguagem Natural utilizando Python e spaCy.

## Funcionalidades

- Análise de tokens
- POS Tagging (classe gramatical)
- Dependência sintática
- Geração de árvore sintática
- Detecção de expressão multipalavra (MWE)

## Exemplo

Frase analisada:

"O analista de PCP quebrou um galho para o desenvolvedor."

## Tecnologias

- Python
- spaCy

pip install spacy
